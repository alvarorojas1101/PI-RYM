Proyecto Integrador Rick and Morty
==================================

¡Bienvenido al repositorio del Proyecto Integrador Rick and Morty!

Este proyecto fue desarrollado como parte del programa de formación en desarrollo web en SoyHenry. Se trata de una aplicación web centrada en la temática de Rick and Morty, que permite a los usuarios explorar y gestionar una base de datos de personajes de la serie.

Características Principales
---------------------------

*   **Autenticación de Usuarios:** Los usuarios pueden crear cuentas y acceder a la aplicación utilizando credenciales seguras.
*   **Exploración de Personajes:** Los usuarios pueden buscar, filtrar y explorar una extensa base de datos de personajes de Rick and Morty.
*   **Gestión de Favoritos:** Los usuarios pueden agregar personajes a una lista de favoritos y eliminarlos según sus preferencias.
*   **Interfaz Intuitiva:** Diseño atractivo y receptivo para una experiencia de usuario óptima.

Tecnologías Utilizadas
----------------------

*   JavaScript
*   React
*   Node.js
*   Redux
*   HTML
*   CSS
*   PostgreSQL
*   Express.js
*   Sequelize

Instalación y Uso
-----------------

1.  Clona este repositorio en tu máquina local.
2.  Instala las dependencias utilizando `npm install`.
3.  Inicia el servidor backend con `npm start`.
4.  Inicia el servidor frontend con `npm run dev`.
5.  Abre tu navegador y accede a `http://localhost:3001`.
